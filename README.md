## ✨Features

- user can give like to the images
- user can filter the images
- user can see the images based on Category

## 🚀Installation
1. ``` git clone https://github.com/jaisuriya97/Dribbble_Clone/ ```
2. ``` npm install ```
3. ``` npm start ```
